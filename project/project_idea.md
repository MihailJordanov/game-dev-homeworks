## Project Idea - Yonedrfall

1. The game is a 2D metroidvania where the player explores an interconnected world and unlocks new areas through abilities.
2. The player can move, jump, crouch and fight enemies.
3. New abilities like double jump, dash and wall jump will unlock progression paths.
4. The world will include hidden areas, moving platforms, one-way platforms and hazards.
5. There will be different enemy types with basic AI.
6. The game will have a health system.
7. Save/load and progression systems will be included.
8. The game will feature boss battles.
9. A basic UI with HUD, menu and game over screen will be implemented.
10. Polish elements like sound and visual effects will be added.
